# My Crafty Stash — Apple (Mac + iPad)

Native SwiftUI companion to the [My Crafty Stash](https://github.com/TequilaJosh/MyCraftyStash)
Windows desktop app. One multiplatform target runs on macOS 14+ and iPadOS 17+.

**Phase 1 (current): read-only cloud stash browser.** Sign in with an `mcs_`
API key generated by the Windows app (Settings › Cloud Sync › API Keys) and
browse the inventory that the desktop synced to the cloud. See
[docs/PORT-PLAN.md](docs/PORT-PLAN.md) for the roadmap to a full peer app.

## Building

There is no `.xcodeproj` in the repo — it's generated:

```bash
brew install xcodegen
xcodegen generate
open MyCraftyStash.xcodeproj
```

No Mac handy? Every push to `main` compile-checks both destinations
(macOS + iPad Simulator) on GitHub's macOS runners — see
`.github/workflows/build.yml`. That workflow is the source of truth for
"does it build" until real hardware arrives.

## Architecture notes

- **No shared code with the Windows app** — this is a Swift rewrite. What IS
  shared: the cloud API (Azure Static Web Apps at
  `wonderful-meadow-0a1a40110.7.azurestaticapps.net`), the data shapes, and
  the product design.
- Auth: same `mcs_` API keys as the desktop, sent as `X-Api-Key`, stored in
  the Keychain (the desktop's DPAPI equivalent).
- `Sources/Models` mirrors the API JSON; dates stay ISO-8601 strings because
  .NET's fractional seconds break Foundation's `.iso8601` decoding strategy.
